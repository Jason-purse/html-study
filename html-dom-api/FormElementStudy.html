使用注意事项

获取一个表单元素

你能够通过Css选择器 - 通过querySelector方法进行抓取,或者通过在当前文档中的forms属性获取到所有的表单列表 ..
此属性返回一个HTMLFormElement对象的数组 - 列出在当前页面的所有表单 ... 你能够使用以下的语法去获取单独的一个表单

1.document.forms[index]
返回数组中指定的某一项
document.forms[id]
返回ID是 id的表单
3.document.forms[name]
返回表单name attribute的值为 name的表单 ..

访问表单的元素

通过检查表单的elements属性来获取表单的包含数据的元素列表, 这将返回一个HTMLFormControlsCollection 它列出了表单的用户数据项的元素列表 .. 这些都是`form`的后代，以及那些使用表单属性成为表单成员的成员。 你能够通过使用它们的name属性作为form的key来获取表单元素,通过使用elements是最棒的方式.. 它仅仅包含了表单的元素,它不会与表单的其他属性混合 ..
